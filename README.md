# 🎵 Vibes Radio PC

Application web de streaming radio pour PC avec animations saisonnières immersives.

## 💻 Aperçu

Vibes Radio PC est une expérience d'écoute radio optimisée pour grand écran avec :
- **38 radios internationales** de 8 pays
- **Interface immersive plein écran** avec animations saisonnières
- **4 thèmes saisonniers** : Hiver ❄️, Printemps 🌸, Été ☀️, Automne 🍂
- **Easter Eggs interactifs** cachés dans l'interface
- **Design glassmorphism** moderne et élégant

## ✨ Fonctionnalités

### 📻 Radios Internationales
- **38 stations** actives de qualité
- **8 pays** : France, USA, UK, Allemagne, Espagne, Italie, Japon, Brésil
- Streaming haute qualité
- Contrôles intuitifs (Play/Pause)
- Interface épurée et moderne

### 🌈 Thèmes Saisonniers
- **Changement automatique** selon la date
- **Changement manuel** avec triple-clic sur les emojis
- **Animations immersives** pour chaque saison :
  - ❄️ **Hiver** : Flocons de neige tombants
  - 🌸 **Printemps** : Fleurs flottantes avec papillons
  - ☀️ **Été** : Soleil pulsant avec rayons
  - 🍂 **Automne** : Feuilles tourbillonnantes

### 🎮 Easter Eggs
**Easter Eggs cachés** à découvrir :
- 🎄 **Triple-clic sur ❄️** : Animation spéciale Hiver
- 🌸 **Triple-clic sur 🌸** : Animation spéciale Printemps
- ☀️ **Triple-clic sur ☀️** : Animation spéciale Été
- 🍂 **Triple-clic sur 🍂** : Animation spéciale Automne
- 🎁 Et d'autres secrets à découvrir...

### 🎨 Design
- **Interface glassmorphism** avec effets de blur
- **Animations fluides** et performantes
- **Responsive** : S'adapte à toutes les tailles d'écran
- **Mode plein écran** recommandé pour l'expérience optimale
- **Particules saisonnières** en arrière-plan

## 🚀 Technologies

- **HTML5** - Structure
- **CSS3** - Animations et glassmorphism
- **JavaScript Vanilla** - Logique et interactions
- **HTML5 Audio API** - Streaming radio
- **Canvas API** - Animations de particules

## 📦 Installation

### 🌐 En ligne
Accède directement à l'application :
```
https://tfasty.github.io/winter-vibes-radio/
```

### 💻 Installer comme application (PWA)

**Chrome / Edge / Brave :**
1. Ouvre l'application dans ton navigateur
2. Clique sur l'icône **💻 Installer** dans la barre d'adresse
3. Clique sur **"Installer"**
4. L'app s'installe sur ton bureau ! 🎉

**Ou via le menu :**
- Chrome/Brave : Menu **⋮** > **"Installer Vibes Radio PC..."**
- Edge : Menu **⋯** > **"Applications"** > **"Installer ce site"**

**Avantages de l'installation :**
- 🚀 Lancement rapide depuis le bureau
- 🖥️ Fenêtre dédiée (sans barre de navigation)
- ⚡ Performances optimisées avec cache
- 💾 Fonctionne même hors ligne (interface uniquement)

📖 **[Guide d'installation détaillé](INSTALLATION_GUIDE_PC.md)**

### 🔧 Local
1. Clone le repository :
```bash
git clone https://github.com/TheFasty/winter-vibes-radio.git
```

2. Ouvre `index.html` dans ton navigateur

3. Appuie sur **F11** pour le mode plein écran (recommandé)

## 🎯 Utilisation

### Navigation
- **Sélection radio** : Clique sur une station dans la liste
- **Play/Pause** : Bouton central ou Espace
- **Changer de saison** : Triple-clic sur les emojis saisonniers (❄️🌸☀️🍂)
- **Mode plein écran** : F11 (recommandé)

### Raccourcis clavier
- **Espace** : Play/Pause
- **F11** : Mode plein écran
- **Esc** : Quitter le plein écran

### Easter Eggs
- **Triple-clic** sur les emojis de saison pour débloquer des animations spéciales
- Explore l'interface pour découvrir d'autres secrets !
